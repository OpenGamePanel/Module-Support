How to install Uprise:

1. Upload all files to the root folder of your OpenGamePanel installation and overwrite old files.

2. Go to Modules at /home.php?m=modulemanager and install the module by clicking Install next to Support.

A new tab called Support has now been added to your navigation bar.
The tab is visible to all OGP users (admins, users and sub users).
